완성된 사이드바 테스트

사이드바 메뉴를 클릭해서 하위 메뉴가 열리는지 확인하세요.


테스트 체크리스트:

  • 시험 관리 클릭 → 하위 메뉴 열림/닫힘
  • 라이선스 클릭 → 하위 메뉴 열림/닫힘
  • 화살표 아이콘 180도 회전
  • 다른 메뉴 클릭 시 이전 메뉴 닫힘
  • 페이지 로드 시 시험 관리 자동 열림
