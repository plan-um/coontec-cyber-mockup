사용자 시나리오

쿤텍 사이버 복원력 서비스의 주요 사용자 흐름

웹화면

신규 고객사 담당자 가입 및 로그인

  1. 회원가입 페이지에서 정보를 입력하고 가입을 완료합니다.
  2. 로그인 페이지에서 계정 정보로 시스템에 접속합니다.
  3. 메인 대시보드에서 서비스 전체 현황을 한눈에 파악합니다.
웹화면

관리자의 신규 프로젝트 생성 및 시험 준비

  1. 통합 프로젝트 메뉴에서 새 프로젝트 생성을 시작합니다.
  2. 선박 정보를 입력하거나 NBM 연동으로 프로젝트를 생성합니다.
  3. '관리영역(Zone)' 설정에서 시험 대상 IP 범위를 지정합니다.
웹화면

운영자의 스캔시험 설정 및 결과 확인

  1. 스캔시험 메뉴에서 대상 Zone을 선택하고 시험 설정을 시작합니다.
  2. 스캔 옵션을 지정하고 시험을 실행합니다.
  3. 완료 후 결과 페이지에서 취약점을 분석하고 CSV로 내보냅니다.
웹화면

관리자의 라이선스 발급 및 할당

  1. 라이선스 관리 메뉴에서 신규 라이선스를 발급합니다.
  2. 요청/할당 메뉴에서 라이선스 요청을 승인 처리합니다.
  3. '사용자 할당' 기능으로 실무자에게 라이선스를 배정합니다.
웹화면

시스템 관리자의 사용자 활동 로그 감사

  1. 감사/로그 메뉴로 이동하여 사용자 활동을 추적합니다.
  2. 기간이나 활동 유형별로 로그를 필터링하여 조회합니다.
  3. 주요 활동 내역을 확인하고 필요시 CSV로 다운로드합니다.
어플리케이션

현장 엔지니어의 오프라인 로그인 및 인증

  1. 앱을 실행하고 ID/PW 또는 생체인증으로 로그인합니다.
  2. 라이선스 관리에서 '오프라인 인증'을 선택합니다.
  3. 인증코드를 QR 스캔 또는 수동 입력하여 라이선스를 활성화합니다.
어플리케이션

현장 엔지니어의 프로젝트 생성 및 Zone 설정

  1. 프로젝트 관리에서 새 프로젝트 생성을 시작합니다.
  2. 선박 정보를 수동으로 입력하여 프로젝트를 만듭니다.
  3. 생성된 프로젝트의 '네트워크 존'에서 시험할 IP 대역을 설정합니다.
어플리케이션

현장 엔지니어의 부하시험 및 사설 보고서 생성

  1. 네트워크 존에서 '부하시험'을 선택하고 설정을 지정합니다.
  2. 시험을 실행하고 결과(성공/실패, 응답시간)를 분석합니다.
  3. '사설 보고서' 기능으로 현장에서 즉시 PDF 보고서를 생성합니다.
어플리케이션

현장 엔지니어의 시험 데이터 백업

  1. 데이터 관리의 '백업/복구' 메뉴로 이동합니다.
  2. '백업 실행'을 선택하고 데이터 범위와 옵션을 지정합니다.
  3. 백업을 시작하고 완료될 때까지 진행 상황을 확인합니다.
